# 🧪 JavaScript Básico – Star Wars

Projeto prático desenvolvido durante o curso **JavaScript para QAs** (Udemy),  
ministrado por **Fernando Papito**.

Este repositório tem como objetivo registrar meu aprendizado nos fundamentos
de **JavaScript aplicados à área de Qualidade de Software (QA)**, utilizando
exercícios e exemplos práticos com temática **Star Wars**.

Ao longo do projeto, foi desenvolvida uma **WebApp simples utilizando Vue.js**,
permitindo praticar lógica de programação e manipulação de dados no JavaScript.

---

## 🎯 Objetivo do Projeto

Construir uma base sólida em **JavaScript** para:

- Evoluir como QA
- Entender melhor o funcionamento do código
- Desenvolver lógica de programação
- Preparar o caminho para **automação de testes**

---

## ⚙️ Funcionalidades

A aplicação possui funcionalidades simples para praticar conceitos da linguagem:

- Listagem de personagens
- Filtro de busca
- Simulação de exclusão de itens
- Renderização dinâmica com **Vue.js (v-for)**
- Estilização com **CSS inspirado em Star Wars**

> Observação:  
> O projeto não utiliza API REST. As funcionalidades foram simuladas com
> manipulação de listas em **JavaScript**, pois o foco do curso é aprender
> a linguagem e sua lógica.
